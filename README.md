# Dashboard-Interactive-python
It is basically an app created on strealit which shows an interactive dashboard with lots of anylisis based on a certain dataset. The data collected tells us about the economy of a buisiness in the United Kingdom back in 2011. The dataset has more than a million rows and collects data from 3 different years from 2009 to 2011. 
There were many different libraries from python used in this project such as pandas , numpy , matplotlib , seaborn , etc.

## Description
The data was first depured into different properties of a single class, it would be able to handle more dataframes if needed. Then i moved on to the graphs, made on matplotlib and seaborn 
they intended to be as clear as they could so you didn't have the need to look at the dataframe to realise what you are looking. plot_dark_style is a file with all the rcparams needed so that the graphs shown in the app wouldn't look to bad, there i change settings like the axes facecolor , the fontsizes and the color in which they were showed. To finish the project i planned and coded the whole website on streamlit so that it wouldn't take me to long to create the frontend needed to show this anylisis.
